# Garvis Command Center

גרסה ראשונה לאייפון כ-Web App.

מה יש בפנים:
- מסך פקודה אישי
- בחירת מצב עבודה
- בחירת סוכן
- בחירת רמת אישור
- ניתוח ראשוני של המשימה
- הכנת פרומפט לסוכן המתאים
- שמירת הפקודה האחרונה בדפדפן

איך לפתוח באייפון:
1. העלה את התיקייה לשרת קטן, GitHub Pages, Netlify, Vercel או כל אחסון סטטי.
2. פתח את הקישור דרך Safari באייפון.
3. לחץ Share.
4. בחר Add to Home Screen.
5. עכשיו Garvis יופיע כאפליקציה במסך הבית.

זו גרסה מקומית ללא API. בשלב הבא אפשר לחבר אותה לשרת אמיתי ולמודלים כמו ChatGPT, Claude, Grok ו-Codex.
